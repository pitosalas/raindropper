# Raindropper Project Memory

## Always use `uv` to run commands
- Run tests: `uv run pytest tests/ -v`
- Never use `.venv/bin/python` directly

## Project structure
- Source: `raindropper/` (actions.py, client.py, main.py, selection_set.py)
- Tests: `tests/`
- Features: `.j2/features/features.md`
- State: `.j2/state.md`

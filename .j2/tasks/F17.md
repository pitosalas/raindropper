# F17 — Interactively Delete Tags in Selection Set in Groups of 10

## T1 — Add `delete_tags_interactively` to `actions.py`
**Status**: done

Add a new function `delete_tags_interactively(client, selection_set)` to `raindropper/actions.py`.

- If selection set is empty, print "Selection set is empty." and return.
- If `selection_set.kind != "tags"`, print "Selection set does not contain tags." and return.
- Loop over tags in batches of 10 using `range(0, len(tags), 10)`.
- For each batch, print `Tags N–M of Total:` then numbered tag names.
- Prompt: `Delete (numbers, 'a'=all, Enter=skip, 'q'=quit): `
- Parse input: `'a'` → delete all in batch; numbers → delete those; Enter → skip; `'q'` → stop.
- Call `client.delete_tags([...])` with the selected tag ids for each accepted selection.
- Print running total after each batch that has deletions.
- After all batches, print: `Deleted N of M tag(s).`

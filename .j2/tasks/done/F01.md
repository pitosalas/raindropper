# Tasks for F01 — Raindrop API Client

### T01 — Set up project with uv
**Status**: done
**Description**: Initialize the project using `uv init`, create the `raindropper/` package subfolder, and add `requests` as a dependency. Ensure `pyproject.toml` has a `raindropper` entry point pointing to `raindropper.main:main`.

### T02 — Implement RaindropClient
**Status**: done
**Description**: Create `raindropper/client.py` with a `RaindropClient` class. Read `RAINDROP_TOKEN` from the environment on init (crash if missing). Implement `fetch_tags()` that calls `GET /tags` and returns the parsed JSON list.

### T03 — Write tests for RaindropClient
**Status**: done
**Description**: Create `tests/test_client.py`. Use `unittest.mock.patch` to mock HTTP responses. Test that `fetch_tags()` returns the expected list and that missing `RAINDROP_TOKEN` raises a `KeyError`.
